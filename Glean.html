GLEAN APP

The word GLEAN means to gather. Glean is a simple and effective to-do app that helps you get things done. With Glean, you can create tasks, set deadlines, and track your progress. Glean is perfect for anyone who wants to stay organized and productive.

FEATURES

  1. ADD

    2. This feature allows you to create your Todos easily


  2. EDIT

    3. This feature enables you to edit Todos allowing to write or rewrite a Todo


  3. DELETE

    4. This feature enables you to be able to remove Todos after completing them.

ABOUT

Glean was inspired by my own need for a way to organize the things I do. I was always finding myself with a to-do list that was too long and overwhelming. I wanted an app that would help me to prioritize my tasks and track my progress. I also wanted an app that was easy to use and didn't have a lot of bells and whistles. I wanted something that I could quickly open and add a task to, without having to spend a lot of time learning how to use it. Glean is the result of my desire to create a simple and effective to-do app that would help people to get things done. I hope you find it helpful! Source code
